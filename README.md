  ## 6. 담당 기능
  - Ajax를 사용하여 검색 키워드와 지역, 직무를 선택하여 다중 검색이 가능하도록 구현
  - 최근 등록한 공고와 좋아요가 많은 순으로 공고를 볼 수 있도록 배치, 로그인 시 관심 있는 공고를 북마크 가능
  - 등록일순 / 좋아요순 / 평점순으로 공고 목록을 정렬 기능
  - 로그인 후 이력서 작성 및 수정이 가능하도록 구현
